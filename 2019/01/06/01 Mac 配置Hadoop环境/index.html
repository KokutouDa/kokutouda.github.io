Mac 配置Hadoop环境

准备启动Hadoop

etc/hadoop/hadoop-env.sh 文件里设置自己的 Java 路径。

命令行输入 bin/hadoop ，如果显示 hadoop 的用法，说明 Java 路径设置对的。

配置无密码的 ssh

与本地建立连接
ssh localhost

设置无密码
ssh-keygen -t rsa -P '' -f ~/.ssh/id_rsa
cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
chmod 0600 ~/.ssh/authorized_keys

执行一个 MapReduce 任务

  1. 格式化文件系统。

    在单一系统中，一个namenode 就是一个文件系统，一个namenode 对应多个 datanode。

    bin/hdfs namenode -format

  2. 打开 NameNode、DataNode 的进程

    sbin/start-dfs.sh

  3. 网页访问 namenode - “http://localhost:50070 “（hadop3.0.0以上版本进入 “http://localhost:9870"）

  4. 创建目录

    bin/hdfs dfs -mkdir /user
    bin/hdfs dfs -mkdir /user/用户名

  5. 把本地文件复制到分布式文件系统

    bin/hdfs dfs -put etc/hadoop input

    「input」是相对路径，绝对路径是在「/user/用户名/…」下。

    tips: bin/hdfs dfs -put [本地目录] [目标目录]

  6. 运行提供的 jar 案例，并计算输出

    bin/hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-2.9.2.jar grep input output 'dfs[a-z.]+'

    tips: hadoop jar <可执行jar文件> grep [源目录][输出目录] ‘正则字符串’`
    字符串：用于过滤需要计算的文件有哪些。

  7. 把输出的目录复制到本地查看
    bin/hdfs dfs -get output output
    cat output/*

    tips: bin/hdfs dfs -get [输出目录] [本地新建目录名]
